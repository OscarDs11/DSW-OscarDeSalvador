Tema 1: Etiqueta Básica de HTML.

HTML: (Conocido como un Lenguaje de Marcas de Hipertexto, del inglés - Hyper Text Markup Language)

Es un lenguaje de marcado. Esto significa que nos permite definir la estructura y contenido de nuestra página web usando etiquetas. Toda parte de una página web se traduce en etiquetas HTML. Si quieres añadir un párrafo, utilizas la etiqueta "p." Si quieres una imagen, utilizas la etiqueta "img." Si quieres agregar un link al sitio web, utilizas la etiqueta "a."

Las etiquetas HTML se construyen de la siguiente forma:

Etiqueta de apertura:

– Un signo de menor-que (<).

– El nombre de la etiqueta (por ejemplo, a) completamente en minúsculas.

– Un signo de mayor-que (>).

Contenido de la etiqueta (por ejemplo, un texto como “Hola Mundo”).

Etiqueta de cierre:

– Otro signo de menor-que y una barra inclinada (

– El nombre de la etiqueta (de nuevo, a).

– Un último signo de mayor-que (>).

Existen 2 tipos de etiquetas:

- De contenido: Son las etiquetas que llevan el texto, las imágenes y todo que se pueda renderizar en el proyecto.

- Contenedoras: Llevan más etiquetas adentro y nos ayudan a llevar la estructura del proyecto.

Una etiqueta por sí sola en realidad NO es muy útil, necesitamos atributos que definan el comportamiento de nuestras etiquetas. Por ejemplo, las etiquetas de imagen (img) necesitan un atributo (src) para indicar la URL de la imagen. Y una etiqueta a necesita el atributo href para indicar la URL del link que quieres agregar.

Tema 2: Atributos y Valores.

El Atributo, siempre se deben encontrar en la etiqueta inicial, son palabras especiales ubicadas dentro de esta etiqueta.

Los atributos son elementos de las etiquetas que pueden proporcionar ciertas características a un texto, imagen etc. (altura, ancho, color). Van insertos dentro de la etiqueta de inicio o apertura los cuales llevan detrás del signo igual (“=”) los valores del atributo.

HTML. No distingue las mayúsculas de las minúsculas, lo que hace que sea necesario cerrar entre comillas dobles.

Valores: los valores son aquellos que siempre debe permanecer encerrados en comillas, ya sean simples o dobles. ej. “imagen.png” Por lo anterior se dice que el Atributo y Valor deben estar unidos por el signo: =
